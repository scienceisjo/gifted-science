# 영재 과학탐구 · Gifted Science Lab

중학교 **과학영재(중1~2)** 를 위한 자기주도형 탐구 웹앱 모음입니다.
교사의 설명 없이도 학생이 **예측 → 시뮬레이터로 확인 → 직접 측정 → 스스로 점검**까지 해낼 수 있도록 만들었습니다.

🔗 **배포 주소:** https://scienceisjo.github.io/gifted-science/

## 자료

| 자료 | 설명 | 링크 |
| --- | --- | --- |
| 🧪 간이 염분계 보정 탐구 | 부력(액체비중계)과 빛의 굴절로 보정곡선을 만들어 미지 용액의 농도를 알아내는 탐구 웹앱. 굴절·임계각·부력·밀도 시뮬레이터와 자동 Word 보고서 내장. | [salinity.html](salinity.html) |
| 🗺️ 보정 탐구 수업 흐름도 | 도입~정리 8개 Phase로 재구성한 영재 수업 흐름도(심화 발문·평가 관점 포함). | [salinity-flow.html](salinity-flow.html) |

## 특징

- **검증된 물리 모델** — 굴절률·밀도·부력 관계식을 문헌값으로 검증
- **완전 자기주도** — 단계별 예측·힌트·자기점검·문제해결 안내
- **자동 저장·보고서** — 입력 내용이 브라우저에 저장되고 Word 보고서로 정리
- **오프라인 친화** — 각 페이지는 단일 HTML 파일

## 사용

별도 설치 없이 위 배포 주소로 접속하거나, 저장소를 내려받아 HTML 파일을 브라우저로 열면 됩니다.
